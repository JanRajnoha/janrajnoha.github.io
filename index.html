ISUF

Bachelor thesis

ISUF is concept of framework for creating information systems for applications built for Windows 10 on Universal Windows Platform. Main advantage of this framework is simplified craetion of modules for application using code frst approach. Framewrk after analyzing model create database table, module page in application and side panes for working with each records.

ISUF has three main parts - Base, Storage and UI. Each part has different usage and manage different part of application logic.

  • Base - Manage all modules registered in applications. Base part also include support classes, attributes, enums and messages for Messenger design pattern and basic services, like LogService for logging message from application
  • Storage - Manage all accesses into database and working with records. This part also include improved manager for modules.
  • UI - Creating application, module pages, side panes, view models, controls and other behavior of application is main reason of UI part. Like others parts, UI has improved module manager for working with UI modules.

ISUF is currently only concept and demonstrate possibilities of creating modular app on UWP. Modules in concept are only pseudo modules, because they can't be loaded dynamically during app run, but could be really simply created and enabled by settings included in app.

The Daily Notes

One of my the biggest projects for Windows 10 and formerly Windows 10 Mobile. The Daily Notes developed from The Daily Tasks, which was app for tracking your daily activities with showing results in calendar in app and with colorful background for each task. App was really bad and in developer eyes breaks many principles of programming. So I started almost from scratch and take only core to new app. The Daily Notes is application focused for organizing your time with activities, to-dos, notes and many other things which was not developed yet. I tried here many functions, like sharing data from app with friends, creating bigger generic library, programming modular-based app or managing user groups for developing beta versions. TDN was primarily focused for mobiles and platform Windows 10 Mobile, which lacked app of this type. End of mobile Windows means for TDN new start with additions of functions and benefits from desktop Windows. The biggest benefit was Fluent Design system, that could give application modern design and made it different.

All of this things above was transformed and generalized into framework and my bachelor thesis, ISUF.

The Daily Notes was created on UWP technology with XML local storage, beta version distributing and modular system.

DVData

Simple application for managing database of movies with mining information from Czechoslovakia movie database. This application is very similar to DVDatabaze, which was created few years before this, but comes with some new features. One of it was completely changed design and much easier working with records. For movie and people cards was added gallery and app was overall faster. For database was used same technology - SQLite.

Application was created with WPF technology and is available only for Windows. Database was stored locally using SQLite technology and for mining information was used XPath.

DVDatabaze

Basic application for managing database of movies with mining information from Czechoslovakia movie database. This was one of my the first projects. We have many DVD movies and sometimes you need to know, what you have, so I decided to create something simple, what makes importing movies easier. You could insert movie by yourself of try online database for any match. But this was not enough for me, so i added thumbnails, which shows after hovering record in database, online rating and checking, if I saw that movie and if I had it. However, this is only for short term problems. With more and more movies there was more and more information about movie to remember, like summary or actors, so I add function for movie and people cards.

I created this app on elementary school with WinForms technology, so code is not any special, but helped me understand many basics principles of programming. For Database was used SQLite and for data mining parsing web pages using XPath.
